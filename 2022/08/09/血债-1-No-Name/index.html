背景故事
血债|#1 No Name
2022-08-09

  马特拉港口那场噩梦，并没有因为那次大爆炸而彻底结束。天启教会的疯子们把病毒作为敲诈的利器，乌克兰，摩洛哥，日本，新加坡……地铁站，商场，甚至是学校，DX-4的乌云久久的飘荡着。伯尔斯公司成立了新 Bioterrorism Security Assessment Alliance，应对各地的生化公共安全问题。美军在各地的军事基地提供人力和物力支持。国际刑警和BSAA的高压打击下，天启教会的势力日渐衰弱。

  一年后，那场灾难的主角们仿佛都人间蒸发了一般。伯尔斯员工墓地的工号墓碑，孤零零的宣告着他们曾经活过。鲍里斯的墓碑，静静躺在主管区最显眼的位子，每年的白蔷薇，是鲍里斯女儿妮娜悼念父亲的礼物。这位年轻的姑娘对父亲的痛苦和对公司的憎恨，让她逃离了这座伤心的地方。她远走他乡，来到了澳大利亚，在布里斯班定居了下来。作为一名哈佛大学的医学博士，她进入了布里斯班最好的私立医院，成为了一名小有名气的外科手术专家。虽然逃离了那里，但每到父亲的忌日，她总会回到美国，在父亲的墓碑旁，放上那束父亲最爱的花。

  布里斯班远远比波士顿热闹，作为澳大利亚的重要港口，煤矿，汽车，工业设备等等这些，从这里起运，中转，靠岸。美国陆军第22号军事基地就在此驻扎。今年5月，布里斯班也遭遇了DX-4的袭击，教会狂热分子向布里斯班地铁站和时代商场各安放了两枚DX-4毒气弹，向澳大利亚政府索要4000万美元的勒索。在美军“赫尔森”防化部队和BSAA的协作下，成功拆除了毒气弹，解除了危机。澳大利亚安全部门逮捕了安放毒气弹的恐怖分子，遗憾的是，主谋扎伊尔并未被逮捕，目前，澳大利亚安全部门与CIA正全力搜捕扎伊尔。

  妮娜从电视上看到了这件事，她只希望这个扎伊尔最好被乱枪打死。她的父亲，就是被这帮疯子杀害的。现在的妮娜，从市区的总院调到了港口的分院，成为港口分院的主管。她很珍惜这次机会，一直兢兢业业工作着。如果说唯一的不愉快，大概只有22区的那个纳尔逊军医官老是缠着自己，她暂时没有那方面的打算。

  转眼来到8月，新闻报道着杂七杂八无关紧要的事，布里斯班港口热火朝天。拜国际经济形势好转，8月份的吞吐量超过去年同时期吞吐量的三倍以上。港口异常的繁忙，港口所有人都在连轴转，为了保证运转，运营公司开始招募大批临时工人，待遇丰厚让许多非法移民心动。不少人通过各种手段混到这份差事。包括已经失去资金来源的扎伊尔。

  扎伊尔不仅仅想的是那份薪水，而是整个布里斯班港，他的手里，还有一枚高剂量的DX-4生物弹。早在一个月以前，他就盯上了港口，现在，他终于可以“净化”这些低等物种了。

  8月25日晚上十点，港口灯火通明，扎伊尔在员工宿舍里释放了那枚生物弹，紧接着，他将DX-4注射进自己的体内。他朝着马拉维的方向朝拜着。两名工友看见他的怪异举动，也只是笑了笑。并不知道他们将看到一场大屠杀。十点十五分，员工宿舍发出了瘆人的惨叫声。正在码头上工作的工人们看见，一群衣衫不整的怪物正冲向他们，疯狂撕咬着他们身上的每一处地方。港口的保安顿时吓得魂不附体，开了两枪便逃之夭夭。

  等到布里斯班警察总局陆续接到港口附近居民的报警电话说有怪物闯入时，已经是26日凌晨三点。澳大利亚政府紧急求援美国。布里斯班警察和安全部队体现出难得的责任感，组织营救居民，但丧尸的感染数量不断攀升，只能疏散部分居民，并建立隔离工事，阻止丧尸扩大感染范围。

  第22号基地命令上次处理毒气事件的“赫尔森”防化部队立刻待命，进入一级警备状态。那个纠缠妮娜的军医官就在这支部队服役。根据美军的内部资料显示，该部队指挥官是一名叫列昂尼德·尼基塔·莫洛托夫的俄裔美国人，但五角大楼都没有此人一张清晰的照片。在基地，他总是带着面罩或者是防冲击面具。因此，在基地有得了一个外号——面具老爹。如果辨认他，就是那口带有德国东部口音的英语和臂章上的白蔷薇，臂章上的白蔷薇面具老爹解释说，那是他过世妻子最爱的花，他妻子是个英国人。

  8月26日下午五点，早已经准备完毕的“赫尔森”防化部队接到了五角大楼和白宫的命令——不惜一切代价消灭这次生化事故并营救感染区居民。军需官将DX-4抑制剂分发到每个医疗兵手中。面具老爹给了每个人10分钟时间做最后一遍检查。军医官纳尔逊机械的检查着携带装备，脑海里全是妮娜。事故发生后，妮娜工作的医院位于感染区，而妮娜当天负责夜班。他尝试联系妮娜，可电话始终处于无人接通状态。他默默祈祷着，不要有事。

  在路上，面具老爹拿出作战地图，与参谋们分析着制定的预案——他们要先深入港口，港口有一支当地的特警小队，小队长弗莱明熟悉感染区的具体情况。由他带路，根据无人机侦察的情况，前往固定的撤离点撤离感染区居民。撤离完毕后由轰炸机投放中子弹进行生物消杀。

  早在26号上午，电视台直升机在感染区上空用广播通知居民前往撤离点撤离。此时，在车中的纳尔逊，此时只希望妮娜听到了广播，前往撤离点。

  面具老爹不知道妮娜，他只知道，天启教会那帮王八蛋，要血债血偿！
